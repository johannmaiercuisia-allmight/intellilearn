## Introduction

Intellilearn is an AI-powered Learning Management System (LMS) designed to facilitate online education. The system supports three user roles: Admin, Instructor, and Student, each with specific capabilities and permissions.

---

## Getting Started

### Logging In

1. Navigate to the Intellilearn login page
2. Enter your email and password
3. Click "Login"

### First-Time Login

If this is your first time logging in:
- Check your email for verification link (if required)
- Update your profile information
- Change your default password

## User Roles Overview

Intellilearn has three distinct user roles:

| Role | Primary Purpose | Access Level |
|------|----------------|--------------|
| **Admin** | System management and user administration | Full system access |
| **Instructor** | Course creation and student management | Course-level access |
| **Student** | Learning and assessment participation | Enrolled course access |

---

## Admin Role

Administrators have full control over the system and can manage all users, courses, and content.

### Admin Capabilities

#### User Management
- **View All Users**: See complete list of all system users
- **Create Users**: Add new students, instructors, or admins
  - Set user role during creation
  - Assign initial credentials
- **Edit Users**: Modify user information
  - Change user roles
  - Update contact information
  - Activate/deactivate accounts
- **Delete Users**: Remove users from the system

#### Course Management
- **View All Courses**: Access to every course in the system
- **Create Courses**: Set up new courses
  - Assign course code and name
  - Set semester and section
  - Assign instructor
  - Add course description
- **Edit Courses**: Modify course details
- **Delete Courses**: Remove courses from system
- **View Course Analytics**: Monitor enrollment and activity

#### System Oversight
- **Monitor All Activities**: View system-wide activity
- **Access All Content**: View all lessons, assessments, and materials
- **Manage Enrollments**: Enroll or unenroll students from courses
- **View All Grades**: Access student performance data

### Admin Dashboard Features
- User statistics overview
- Course enrollment metrics
- System activity logs
- Quick access to user management

## Instructor Role

Instructors manage their assigned courses, create content, and evaluate student performance.

### Instructor Capabilities

#### Course Management
- **View My Courses**: See all courses assigned to you
- **Edit Course Details**: Update course information
  - Modify description
  - Update semester/section info
- **View Enrolled Students**: See student roster
- **Manage Enrollments**: Approve or remove student enrollments

#### Lesson Management
- **Create Lessons**: Add new learning content
  - Set lesson title and description
  - Define lesson order
  - Set publish status
- **Upload Materials**: Add learning resources
  - PDF documents
  - Video files
  - Presentations
  - Links to external resources
- **Edit Lessons**: Modify existing content
- **Delete Lessons**: Remove lessons and materials
- **Organize Content**: Arrange lessons in logical order

#### Assessment Management
- **Create Assessments**: Design various assessment types
  - Quizzes
  - Long Exams
  - Individual Activities
  - Group Activities
  - Recitations
- **Configure Assessment Settings**:
  - Set time limits
  - Define maximum attempts
  - Set availability dates
  - Assign point values
  - Publish/unpublish assessments
- **Create Questions**: Add assessment items
  - Multiple Choice
  - True/False
  - Short Answer
  - Essay
  - Set correct answers and points
- **Bulk Import Questions**: Upload multiple questions at once
- **Edit Questions**: Modify existing questions
- **Delete Questions**: Remove questions from assessments

#### Grading & Evaluation
- **View Submissions**: See all student submissions
- **Auto-Grading**: System automatically grades:
  - Multiple choice questions
  - True/false questions
- **Manual Grading**: Grade subjective responses
  - Short answer questions
  - Essay questions
  - Provide feedback comments
- **Grade Management**: 
  - View student grades
  - Compute final grades
  - Export grade reports

#### Communication
- **Create Announcements**: Post course updates
  - Set announcement priority
  - Schedule announcements
- **Edit Announcements**: Modify existing posts
- **Delete Announcements**: Remove announcements

#### Calendar Management
- **Create Events**: Add course-related events
  - Assignment due dates
  - Exam schedules
  - Class sessions
- **Edit Events**: Modify event details
- **Delete Events**: Remove events

### Instructor Dashboard Features
- Course overview cards
- Recent student activity
- Upcoming assessments
- Pending grading queue
- Quick access to course materials

## Student Role

Students access course content, complete assessments, and track their academic progress.

### Student Capabilities

#### Course Access
- **View Enrolled Courses**: See all courses you're enrolled in
- **Enroll in Courses**: Join available courses
- **Unenroll from Courses**: Leave courses (if permitted)
- **View Course Details**: Access course information and syllabus

#### Learning
- **Access Lessons**: View published lesson content
- **Download Materials**: Get course resources
  - PDFs
  - Videos
  - Presentations
  - Additional files
- **Track Progress**: Monitor lesson completion
- **Mark Lessons Complete**: Update your progress status

#### Assessments
- **View Available Assessments**: See published quizzes and exams
- **Start Assessment**: Begin taking an assessment
  - View time limit
  - See attempt count
  - Review instructions
- **Submit Answers**: Complete and submit responses
- **View Results**: See graded assessments
  - View score and percentage
  - See correct answers (if enabled)
  - Read instructor feedback
- **Retake Assessments**: Attempt again (if allowed)
- **Track Attempts**: Monitor remaining attempts

#### Grades & Performance
- **View Grades**: See assessment scores
- **Track Overall Performance**: Monitor course progress
- **View Grade Breakdown**: See performance by assessment type
- **Access Grade History**: Review past submissions

#### Communication
- **Read Announcements**: Stay updated with course news
- **View Activity Feed**: See recent course updates
  - New announcements
  - New assessments
  - New materials uploaded

#### Calendar
- **View Course Events**: See upcoming deadlines
- **Track Due Dates**: Monitor assignment schedules
- **View Exam Schedules**: See test dates

### Student Dashboard Features
- Enrolled courses overview
- Activity feed with recent updates
- Upcoming deadlines
- Recent grades
- Progress indicators
- Quick access to active assessments

## Assessment Types Explained

### Quiz
- Short, focused assessments
- Usually multiple choice or true/false
- Auto-graded
- Multiple attempts often allowed

### Long Exam
- Comprehensive assessments
- Mix of question types
- Time-limited
- Usually one or two attempts

### Individual Activity
- Solo assignments
- May include essays or short answers
- Requires manual grading
- Flexible deadlines

### Group Activity
- Collaborative assignments
- Shared submissions
- Team-based evaluation

### Recitation
- Oral or written participation
- Quick response format
- Frequent, low-stakes assessments

---

## Question Types Explained

### Multiple Choice
- One correct answer from several options
- Auto-graded by system
- Immediate feedback available

### True/False
- Binary choice questions
- Auto-graded by system
- Quick assessment format

### Short Answer
- Brief written responses
- Manually graded by instructor
- Requires instructor feedback

### Essay
- Extended written responses
- Manually graded by instructor
- Detailed feedback provided

## Tips for Success

### For Students
- Check your dashboard daily for updates
- Complete lessons before attempting assessments
- Review feedback on graded work
- Monitor upcoming deadlines in the calendar
- Download materials for offline study

### For Instructors
- Publish lessons in logical sequence
- Set clear due dates for assessments
- Provide timely feedback on submissions
- Use announcements to keep students informed
- Regularly review student progress

### For Admins
- Regularly audit user accounts
- Monitor system usage and performance
- Ensure instructors are assigned correctly
- Review course enrollment patterns
- Maintain system security
